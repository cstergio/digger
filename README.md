# Digger – Arcade Game (Python / Pygame)

This project is an arcade-style game inspired by the classic **Digger (1983)**.
It was developed as part of an academic assignment and focuses on clean
software architecture, real-time gameplay, and AI-controlled enemies.

## Features
- Single-player and local co-op (2 players)
- Tile-based underground level
- Enemy AI with pathfinding
- Score and lives system
- Multiple difficulty levels
- Sound effects and music
- Split-screen multiplayer

## Technologies
- Python 3
- Pygame
- Object-Oriented Design
- MVC / DAO / SOLID architectural principles

## Project Structure
digger/client – game source code
digger/shared – game source code
digger/report/ – technical report (PDF)
digger/media/ – gameplay video



## How to Run
pip install pygame
python main.py
